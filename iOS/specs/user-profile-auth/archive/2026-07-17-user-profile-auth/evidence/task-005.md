# Итог

Реализован автомат состояний флоу авторизации с семантическим значением состояния, типизированными действиями и политикой единственной активной задачи. Модель представления владеет структурированной задачей с монотонным счётчиком поколения для исключения поздних изменений. Локальная проверка выполняется до сетевого запроса.

# Проверки

Команда тестирования пакета: 33 теста, 0 провалов.

- Пустая почта показывает предупреждение — прошёл
- Некорректная почта показывает предупреждение — прошёл
- Существующая почта переходит к входу — прошёл
- Новая почта переходит к регистрации — прошёл
- Пустой пароль показывает предупреждение — прошёл
- Возврат сохраняет почту — прошёл
- Переключение видимости пароля работает — прошёл
- Серверная ошибка показывает обратную связь — прошёл

- Изоляция: модель работает в контексте actor, передаваемые границы
- Отмена: уход отменяет задачу, отмена не меняет состояние
- Генерация защищает от поздних изменений
- Граница приложения: модель не импортирует интерфейс
- Сборка пакета: успешна
- Публичный контракт: модель закрыта за протоколами
- Граф зависимостей: направление от представления к домену
