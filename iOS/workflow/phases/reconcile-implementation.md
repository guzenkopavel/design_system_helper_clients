# iOS addendum: Reconcile Implementation

Use the iOS adapter production roots and protected/excluded roots. Discover the
current Xcode project/workspace, scheme, destination and focused test command
from repository/package evidence; never invent them. Compare Swift/SwiftUI and
project-file changes against the package's selected iOS architecture,
concurrency, testing, accessibility and design-system rules only when those
scopes apply.

Evidence must record the exact finite focused command and result for the
explicit iOS paths. Reconciliation remains package-only: it cannot edit Swift,
project files, shared product artifacts or Android state.
